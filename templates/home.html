Rotadyne client portal

This is a low priority project that I started for a friend of mine around a year ago. I realized that django was going be a better fit and the project stalled while I started down that learning path.

The brief: Build an order tracking website for high volume customers initially, with the view to add in repeat order functionality (B2B transactions) in the future.

I am half way through CS50 web (which teaches django), however I have been side tracked with another opportunity to work on an embedded project. Lately I have been learning C, and the in's and outs of developing embedded applications on microcontrollers from ST Microelctronics.

The project in question aims to deploy a meshed network of LoRa connected devices that are able to detirmine amongst themselves which node is able to activate its drive motor, given the state of all of the nodes sensors in the network. Only one motor may be running in the network at any given time.
